# SMT 表单采集确认表

> 用途：确认每张表单的数据来源、是否门禁、是否签字、是否允许补录，为 MES 采集方案与流程门禁设计提供依据。
> 填写方式：在“数据来源/门禁/签字/补录/设备系统来源”列填写或勾选即可。

---

## 1. 全局确认（必填）

1. 这些表单是否全部需要在 MES 中落地？（是/否）
2. 时间规则：
   - 印锡→回流焊 < 4h：必须拦停 / 只提醒
   - 锡膏暴露 < 24h：必须拦停 / 只提醒
   - 若允许豁免，谁可批准？（岗位/角色）
3. 签字要求：
   - 哪些表单必须签字？
   - 是否必须三方签（生产/技术/质量）？
4. 条码与关联：
   - 工单号/批次号/序列号/物料批次/锡膏批次是否都可扫码？（是/否）
5. 数据来源策略：
   - 允许先人工录入、后续再接设备/系统吗？（是/否）

## 3. 说明

- 如果不确定能否集成设备或系统，请先填“人工录入”，后续再讨论自动化方式。
