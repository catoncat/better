# SMT 表单采集确认表

> 用途：确认每张表单的数据来源、是否门禁、是否签字、是否允许补录，为 MES 采集方案与流程门禁设计提供依据。
> 填写方式：在“数据来源/门禁/签字/补录/设备系统来源”列填写或勾选即可。

---

## 1. 全局确认（必填）

1. 这些表单是否全部需要在 MES 中落地？
   - 否：部分表单明确“不在 MES”（环境温湿度、AOI 点检、生产异常、X-ray），另有流程表已取消（改流程扫描）。
2. 时间规则：
   - 印锡→回流焊 < 4h：只提醒（更准确描述为“回流焊/AOI → 水洗”窗口；水洗机前需扫码点触发校验）
   - 锡膏暴露 < 24h：只提醒
   - 若允许豁免：厂长
3. 签字要求：
   - 哪些表单必须签字：基本上以审批账号为准，除了首件（QR-Pro-05），其余不需要签字/放行
   - 是否必须三方签：不需要
4. 条码与关联：
   - 工单号/批次号/序列号/物料批次/锡膏批次：都可扫码
5. 数据来源策略：
   - 允许先人工录入、后续再接设备/系统：允许

## 3. 说明
