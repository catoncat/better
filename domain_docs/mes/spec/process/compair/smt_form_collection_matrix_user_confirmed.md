- **补录**：不允许

### 印锡→回流焊 < 4h：只提醒
"提醒：
印锡到回流焊是连续过程，工程描述有错误。
工艺上需要控制的是，特定的产品需要在回流焊完成后的4小时内进入“水洗工艺，所以在”水洗工艺“的机器前需要增加扫描仪，当该批产品完成回流焊或AOI（一般直接取AOI的机器时间）计算，超过4小时或特定时间没有进入”水洗机的扫描仪就当time out--提醒"
