## 确认事项

分类,确认事项,确认结果 / 业务标准,备注 / 详细逻辑
系统落地,表单是否全部在 MES 落地？,流程化集成,相关表单不作为独立表单，而是合并在 MES 的工序流程步骤中。
准备门禁,准备完成判定,所有准备项达标（PASS/WAIVED）,门禁=No 允许继续但提醒/豁免并记录；记录必须存在，有可靠自动数据源可自动确认，否则必须人工确认。
时间规则,回流焊 → 水洗时间控制,提醒（Alert，需可豁免；仅适用于配置水洗工序的路由/产品）,业务逻辑修正：原“印锡→回流焊”描述有误。实际为回流焊/AOI 完成后 4h 内需进入水洗工艺。按路由/工序配置，无水洗节点不触发。水洗机前需增设扫描仪，超温自动提醒。
时间规则,锡膏暴露时间（< 24h）,提醒（Alert）,超过 24 小时系统发出预警提醒。
时间规则,异常豁免权限,复用现有角色并显式授权,优先复用线长/生产主管/质量负责人等角色授予豁免权限；若职责仍不清晰，再考虑新增角色或双人豁免（生产+质量）。
签字要求,表单签字/放行要求,审批账号制,基本上以系统登录/操作账号为准替代物理签字。
签字要求,首件检查签字,需签字,首件检查是唯一明确需要独立签字或放行的环节。
签字要求,是否需要三方会签？,否,不需要生产/技术/质量三方同时签字。
条码关联,条码扫描覆盖范围,全部支持,工单号、批次号、SN、物料批次、锡膏批次均支持扫码关联。
数据策略,数据采集演进路径,先人工，后自动,允许前期通过人工录入过渡，后期再对接设备数采或系统集成。
