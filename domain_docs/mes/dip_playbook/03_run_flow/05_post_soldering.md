# 后焊处理

## 1. 概述
后焊处理是 DIP 产线特有的工序段，包括波峰焊后的一系列精整和保护工序：
- **手工焊接**：补焊波峰焊无法完成的焊点
- **剪脚处理**：修剪元器件引脚
- **三防漆喷涂**：保护电路板防潮、防尘、防腐
- **固化处理**：三防漆固化

## 2. 手工焊接

### 2.1 适用场景
| 场景 | 说明 |
|------|------|
| 补焊 | 波峰焊虚焊/透锡不足的点位 |
| 特殊元件 | 无法过波峰焊的敏感元件 |
| 大热容量 | 大电容/大连接器等 |
| 屏蔽罩 | 金属屏蔽罩焊接 |

### 2.2 手工焊接流程
```
PCB 进站（TrackIn）
    ↓
查看焊接任务清单
    ↓
执行手工焊接
    ↓
自检焊接质量
    ↓
PCB 出站（TrackOut）
```

### 2.3 手工焊接要求
| 要求项 | 标准 |
|--------|------|
| 烙铁温度 | 320-380°C（视焊点大小） |
| 焊接时间 | ≤3 秒（普通焊点） |
| 焊点外观 | 光亮、圆润、无尖刺 |
| 透锡率 | ≥75% |

### 2.4 手工焊接不良
| 不良类型 | 代码 | 说明 |
|----------|------|------|
| 虚焊 | HS_COLD | 焊点不饱满 |
| 过热 | HS_OVERHEAT | 焊盘/元件过热 |
| 拉尖 | HS_SPIKE | 锡丝拉尖 |
| 短路 | HS_SHORT | 相邻焊点桥连 |

## 3. 剪脚处理

### 3.1 剪脚目的
- 剪除过长的元器件引脚
- 防止引脚刺穿保护膜
- 改善外观
- 防止短路风险

### 3.2 剪脚流程
```
PCB 进站（TrackIn）
    ↓
执行剪脚
    ↓
检查剪脚高度
    ↓
清理引脚碎屑
    ↓
PCB 出站（TrackOut）
```

### 3.3 剪脚要求
| 要求项 | 标准 |
|--------|------|
| 剪脚高度 | 焊点上方 0.5-1.0mm |
| 剪切面 | 平整无毛刺 |
| 无损伤 | 不损伤焊点 |

### 3.4 剪脚方式
| 方式 | 说明 | 适用场景 |
|------|------|----------|
| 手工剪脚 | 斜口钳手工剪 | 少量/异形 |
| 自动剪脚机 | 设备自动剪 | 批量生产 |

## 4. 三防漆喷涂

### 4.1 三防漆作用
- **防潮**：防止水汽侵入
- **防尘**：防止灰尘附着
- **防腐**：防止化学腐蚀
- **绝缘**：增加绝缘性能

### 4.2 喷涂方式
| 方式 | 说明 | 特点 |
|------|------|------|
| 手工喷涂 | 喷枪手工喷涂 | 灵活但一致性差 |
| 自动喷涂 | 选择性喷涂机 | 一致性好但投资高 |
| 刷涂 | 毛刷涂覆 | 简单但效率低 |
| 浸涂 | 整板浸入 | 覆盖全面但需遮蔽 |

### 4.3 喷涂流程
```
PCB 进站（TrackIn）
    ↓
检查是否需要遮蔽
    ↓
执行喷涂
    ↓
检查喷涂质量
    ↓
PCB 出站（TrackOut）→ 固化
```

### 4.4 喷涂要求
| 要求项 | 标准 |
|--------|------|
| 涂层厚度 | 25-75μm（视产品要求） |
| 覆盖范围 | 按工艺图纸 |
| 禁喷区 | 连接器、测试点、按键等 |
| 外观 | 均匀无气泡、无流挂 |

### 4.5 禁喷区处理
某些区域不能喷涂三防漆：
- 连接器接口
- 测试点/ICT 焊盘
- 按键/开关
- 散热片
- 可调元件

处理方式：
- 预贴遮蔽胶带
- 选择性喷涂设备避开
- 后续清理

### 4.6 喷涂不良
| 不良类型 | 代码 | 说明 |
|----------|------|------|
| 漏喷 | CC_MISS | 应喷区域未覆盖 |
| 气泡 | CC_BUBBLE | 涂层有气泡 |
| 流挂 | CC_SAG | 涂层流挂堆积 |
| 误喷 | CC_WRONG | 喷涂到禁喷区 |
| 厚度不足 | CC_THIN | 涂层过薄 |

## 5. 固化处理

### 5.1 固化方式
| 方式 | 说明 | 时间 |
|------|------|------|
| 室温固化 | 自然晾干 | 24-48 小时 |
| 热风固化 | 烘箱加热 | 15-60 分钟 |
| UV 固化 | 紫外线照射 | 数秒-数分钟 |

### 5.2 热风固化参数
| 参数 | 典型值 |
|------|--------|
| 温度 | 60-80°C |
| 时间 | 30-60 分钟 |
| 升温速率 | ≤5°C/分钟 |

### 5.3 固化流程
```
PCB 进入固化炉（TrackIn）
    ↓
按设定参数固化
    ↓
冷却
    ↓
检查固化效果
    ↓
PCB 出炉（TrackOut）
```

### 5.4 固化检查
| 检查项 | 方法 | 标准 |
|--------|------|------|
| 表干 | 触摸 | 不粘手 |
| 硬度 | 指甲划 | 无痕迹 |
| 附着力 | 划格试验 | 按等级要求 |

## 6. 人工外观检验

### 6.1 检验内容
后焊段完成后，通常需要进行外观检验：

| 检查项 | 标准 |
|--------|------|
| 焊点外观 | 光亮、圆润、无缺陷 |
| 剪脚高度 | 符合工艺要求 |
| 三防漆覆盖 | 均匀、无漏喷 |
| 整体外观 | 无划伤、污染 |

### 6.2 检验流程
```
PCB 进站（TrackIn）
    ↓
目视检查（可配合放大镜）
    ↓
记录检验结果
    ↓
PCB 出站（TrackOut）
    ├─ PASS → 流转测试段
    └─ FAIL → 登记不良 → 返修
```

## 7. 后焊段首件（IPQC）

### 7.1 触发时机
- 批次开始后首块 PCB 进入后焊段
- 换型后首块
- 异常处理后首块

### 7.2 检查内容
| 检查项 | 说明 |
|--------|------|
| 手工焊接质量 | 抽检焊点 |
| 剪脚高度 | 测量确认 |
| 三防漆覆盖 | 对照工艺图 |
| 固化效果 | 检查固化度 |

## 8. 后焊返修

### 8.1 返修场景
| 场景 | 返修方式 |
|------|----------|
| 手工焊接不良 | 重新焊接 |
| 剪脚问题 | 重新剪脚（若过短则报废） |
| 三防漆不良 | 清洗后重喷 |
| 外观缺陷 | 视情况处理 |

### 8.2 三防漆返修
```
清洗（溶剂/机械）
    ↓
干燥
    ↓
重新喷涂
    ↓
重新固化
    ↓
复检
```

## 9. 页面与操作

### 9.1 执行页面
路径：`/mes/execution`

后焊工位操作：
- 选择工位（手工焊接/剪脚/喷涂/固化/外观检验）
- PCB 扫码进站
- 执行作业
- 出站结果录入
- 不良登记
