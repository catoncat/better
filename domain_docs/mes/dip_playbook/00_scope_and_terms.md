# 范围与术语

## 1. 适用范围
本文档集描述 DIP（通孔插件）产线在本系统中的配置与执行流程，覆盖：
- 工单/批次（Run）
- 产线与工位配置（区别于 SMT 的站位概念）
- 插件作业（AI 辅助插件、手工插件、异形件插件）
- 波峰焊接
- 后焊作业（手工焊接、剪脚、三防漆喷涂、固化）
- 首件（FAI）与过程检验（IPQC）
- 测试（ICT/FCT）
- OQC 与完工
- 追溯输出

**不覆盖**：SMT 产线的上料防错、回流焊等流程。

## 2. 角色与职责
| 角色 | 职责 |
|------|------|
| 计划/生管 | 下达工单，创建或释放批次 |
| 工艺工程师 | 配置工位、路由、测试程序、夹具关联 |
| 线长/班组长 | 组织插件作业、首件确认、巡检协调 |
| 操作员 | 执行插件、焊接、后焊、测试等工位作业 |
| 质量（IPQC/OQC） | 首件判定、段首件检查、巡检、抽检与处置 |
| 测试技术员 | ICT/FCT 测试执行、测试程序维护 |
| 设备维护 | 波峰焊设备参数维护、工装夹具 TPM |

## 3. 核心术语与系统实体

| 工单 | WorkOrder | 工单与产品信息 | ERP 同步或工单管理 |
| 路由 | Routing / RouteVersion | 工艺路径与版本 | `/mes/routes` |
| 首件 | FAI | 首件检验任务 | `/mes/fai` |
| 执行记录 | TrackIn/TrackOut | 工位执行事件 | `/mes/execution` |
| OQC | OQC | 出货抽检 | `/mes/oqc` |

## 4. 工位与工段的概念

### 4.1 工位（Station）
DIP 产线的工位是工艺执行位置，用于 TrackIn/TrackOut 执行与数据采集。

与 SMT 的 **站位（FeederSlot）** 概念不同：
- SMT 站位：用于"上料防错"，关注物料加载验证
- DIP 工位：用于"执行追踪"，关注工艺作业完成

### 4.2 工段划分
DIP 产线通常划分为以下工段：

| 工段 | 典型工位 | 说明 |
|------|----------|------|
| 插件段 | AI 插件、手工插件、异形件插件 | 元器件插入 PCB |
| 焊接段 | 波峰焊 | 通孔焊接 |
| 后焊段 | 手工焊接、剪脚、三防漆喷涂、固化 | 焊接后处理 |
| 检验段 | 人工外观检验 | 外观质量确认 |
| 测试段 | ICT、FCT | 电气功能验证 |

## 5. DIP 与 SMT 术语对照

| SMT 术语 | DIP 对应术语 | 说明 |
|----------|--------------|------|
| 站位（FeederSlot） | 工位（Station） | DIP 无站位上料概念 |
| 上料防错 | 物料发料 | DIP 采用领料制而非机台上料 |
| 回流焊 | 波峰焊 | 不同焊接工艺 |
| 贴片 | 插件 | 不同元件安装方式 |
| 钢网（Stencil） | 波峰焊治具 | 不同工装夹具 |
| AOI | 人工外观检验 | DIP 多依赖人工检验 |

## 6. 条码约定
- 单件条码：PCB 序列号，用于追溯
- 物料条码：`物料编码|批次号`
- 工位扫码：工位二维码或手工输入工位码
- 测试设备扫码：设备编码关联测试程序

## 7. 状态定义

### 7.1 Run 状态
| 状态 | 说明 |
|------|------|
| PREP | 准备中，可配置 |
| AUTHORIZED | 已授权，可执行 |
| IN_PROGRESS | 执行中 |
| COMPLETED | 已完成 |
| ON_HOLD | 挂起（OQC 不合格）|
| SCRAPPED | 报废 |

### 7.2 Unit 状态
| 状态 | 说明 |
|------|------|
| QUEUED | 待处理 |
| IN_PROGRESS | 处理中 |
| DONE | 完成 |
| DEFECTIVE | 不良待返修 |
| SCRAPPED | 报废 |
