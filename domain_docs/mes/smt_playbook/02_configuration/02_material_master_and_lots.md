# 物料与批次配置

## 1. 目标
明确物料主数据与物料批次（lot）的管理方式，保证上料防错与追溯可落地。

## 2. 数据如何产生
### 2.1 物料主数据（Material）
- 来源：ERP 同步或物料管理导入。
- 关键字段：
  - `code` 物料编码（Part P/N）
  - `name` 物料名称
  - `unit` 计量单位

### 2.2 物料批次（MaterialLot）
- 来源：上料扫码自动生成或仓库/来料系统导入。
- 关键字段：
  - `materialCode` 物料编码
  - `lotNo` 批次号
- 唯一性：`materialCode + lotNo` 唯一。

## 3. 条码规则与上料关联
- 系统默认识别格式：`物料编码|批次号`。
- 扫码示例：`5212090001|LOT-20250526-001`。
- 上料时会自动创建或复用对应的 `MaterialLot`。

## 4. 数据如何管理
- 物料主数据：由 ERP 维护，MES 只读或有限编辑。
- 物料批次：在上料或来料环节产生，MES 用于追溯与质量记录。

## 5. 真实表单字段对照
在 **SMT物料上机对照表（QR‑Pro‑121）** 中，零件 P/N 就是 `Material.code`。

## 6. 建议的批次号规范
- 推荐格式：`LOT-YYYYMMDD-SEQ`（如 `LOT-20250526-001`）。
- 若供应商有自身批次号，建议保留原批次号并在系统中统一前缀规则。
