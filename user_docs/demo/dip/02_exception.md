# DIP 异常处理演示

> DIP 产线的失败分支与恢复路径。

---

## 概述

DIP 异常处理与 SMT 共享相同的机制，本文档仅列出 DIP 特有的注意事项。

通用异常处理请参考：[SMT 异常处理](../smt/05_exception.md)

---

## 1. Readiness 失败场景

### 1.1 夹具检查失败

**触发**：PREP_FIXTURE 检查失败

**可能原因**：
- 夹具未绑定到产线
- 夹具寿命超限
- 夹具正在维修中

**恢复方式**：

1. **更换夹具**：
   - 选择可用的替代夹具
   - 更新 Run 的夹具绑定
   - 重新执行 Readiness 检查

2. **豁免**（质量确认后）：
   - 使用 quality 账号
   - 点击"豁免"
   - 填写豁免原因

3. **等待维修完成**：
   - 等待维修记录状态变为 VERIFIED
   - 重新执行 Readiness 检查

### 1.2 波峰焊参数未配置

**触发**：路由要求波峰焊参数校验但未配置

**恢复**：
1. 工艺工程师配置波峰焊参数
2. 重新执行 Readiness 检查

## 3. FAI 失败场景

### 3.1 波峰焊质量问题

**常见缺陷**：
- 虚焊
- 锡珠
- 桥接

**恢复流程**：
1. 分析波峰焊参数
2. 调整预热/焊接温度
3. 创建新 FAI
4. 重新试产验证

### 3.2 插件位置偏移

**恢复**：
1. 检查夹具定位
2. 调整插件工艺
3. 重新试产

---

## 4. 执行异常

### 4.1 波峰焊工位异常

**场景**：ST-DIP-WAVE-01 TrackOut FAIL

**常见原因**：
- 焊接温度异常
- 传送速度不匹配
- 助焊剂问题

**处理**：
1. 记录缺陷代码
2. 处置方式：返修/报废
3. 分析并调整工艺参数

### 4.2 功能测试失败

**场景**：ST-DIP-TEST-01 TrackOut FAIL

**处理**：
1. 记录测试失败项
2. 分析失败原因
3. 处置：
   - 可修复 → 返修
   - 不可修复 → 报废

## DIP 特有注意事项

### 工序差异

| SMT 工序 | DIP 对应 | 注意事项 |
|----------|----------|----------|
| SPI 检测 | - | DIP 无 SPI |
| 贴片 | 插件 | 手工/半自动 |
| 回流焊 | 波峰焊 | 参数差异大 |
| AOI | 人工目检/AOI | 可选 |

### 数据采集差异

DIP 常见采集项：
- 波峰焊温度
- 传送速度
- 助焊剂类型
- 预热温度
